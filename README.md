    1. If we use var then we can redeclare a variable which can cause a problem and if let is used then it can't be redeclared, whereas const is used if we don't want to change a value.
    2. Spread operator is used to copy elements from an array which doesn't make any change in the original array.
    3. map() makes a new array by transforming element, filter creates a new array that match a condition whereas forEach() runs a function on each element but it doesn't return anything.
    4. arrow function is like other functions but it is a new features of ES6, shorter way to writing a function.
    5. Template literals is basically (``) a quotation-like sign where we can write html and can use variable inside it
